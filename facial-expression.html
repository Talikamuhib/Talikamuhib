Project Overview

This project implements a Convolutional Neural Network (CNN) for facial expression recognition on the ExpW (Facial Expression in-the-Wild) dataset. The model automatically detects and classifies human emotions from facial images, enabling applications in sentiment analysis, user experience monitoring, and affective computing systems.

Dataset & Methodology

ExpW Dataset Characteristics

  • 91,793 diverse facial images with in-the-wild conditions
  • Multiple ethnicities, ages, and lighting conditions
  • Split: 70% training, 15% validation, 15% testing
  • Data augmentation applied (rotation, flip, zoom) to increase robustness

Architecture: Custom CNN with convolutional blocks, max-pooling layers, and fully connected dense layers. Hyperparameter tuning via grid search and cross-validation optimizes performance on validation data.

Key Applications

Sentiment Analysis

Automatically detect audience sentiment in real-time during presentations, customer interactions, or user testing sessions.

UX Monitoring

Track user emotional responses to interface changes and identify frustration points in user journeys.

Customer Service

Monitor agent-customer interactions for emotional engagement and automatically flag escalation opportunities.

Mental Health

Support therapeutic applications by tracking emotional expressions during counseling or teletherapy sessions.

Research

Enable psychological and social science research on emotional expression patterns and cultural differences.

Accessibility

Help individuals with speech or motor disabilities communicate emotional state non-verbally.

Technical Approach

Training Strategy

  • Architecture: Custom CNN with 4-5 convolutional blocks, dropout regularization, batch normalization
  • Optimizer: Adam with learning rate scheduling for stable convergence
  • Loss Function: Categorical cross-entropy for multi-class classification
  • Regularization: Data augmentation, dropout, early stopping to prevent overfitting
  • Validation: 5-fold cross-validation on training set for robust hyperparameter selection

Limitations & Improvements

  • Model trained on single dataset; generalization to different image sources needs testing
  • Performance may vary with extreme head poses, occlusions, or poor lighting
  • Transfer learning from ImageNet could improve accuracy with limited data
  • Ensemble methods (multiple CNN architectures) could boost robustness
  • Attention mechanisms could provide interpretability (which facial regions drive predictions)
