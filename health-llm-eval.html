Project Overview

Health LLM Eval Harness is a lightweight, production-ready evaluation framework designed for assessing Large Language Models in healthcare contexts. It follows clean code principles with a modular, extensible architecture that minimizes dependencies while maximizing functionality.

Core Features

Pydantic-Based Validation

Strong data type validation and serialization using Pydantic v2, ensuring data integrity throughout evaluation pipelines.

Pluggable Provider System

Extensible provider interface supporting multiple LLM backends (OpenAI, local models, stubs) without modifying core code.

JSONL Dataset Format

Lightweight, streaming-friendly dataset format ideal for large-scale evaluations without memory overhead.

Automated CI/CD

GitHub Actions workflows for continuous testing and deployment, ensuring code quality and reproducibility.

Clean Architecture

Single-responsibility modules with clear separation of concerns: datasets, scoring, checks, and reporting.

Medical Domain Focus

Pre-configured datasets for healthcare evaluation (medical terminology, clinical scenarios) with extensible taxonomy.

Key Components

Evaluation Pipeline

  • Dataset Loading: Stream JSONL files for memory-efficient handling of large evaluation sets
  • Provider Selection: Choose LLM backend (OpenAI, local, stub) via configuration
  • Scoring: Apply domain-specific metrics (accuracy, F1, medical relevance)
  • Validation: Run checks for consistency, bias detection, and output validation
  • Reporting: Generate human-readable reports with results and recommendations
